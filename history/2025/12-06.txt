### Коммит: 52f6e11 - "Add Font Awesome icons and theme toggle button with dark/light mode support"
- **Что сделано**: Добавлены иконки Font Awesome 6 и кнопка переключения темы (светлая/темная) с полной поддержкой тем Bootstrap
- **Как сделано**:
  * Подключен Font Awesome 6.5.1 через CDN в `<head>` (link с integrity и referrerpolicy)
  * Добавлена кнопка переключения темы в `index.html`: `<button>` с классом `btn btn-outline-secondary rounded-circle`, позиционирование через `position-fixed top-0 end-0 p-3`
  * Добавлен метод `toggleTheme()` в Vue.js для переключения между 'light' и 'dark'
  * Добавлен метод `applyTheme()` для применения темы к элементу `<html>` через атрибут `data-bs-theme` (Bootstrap 5 требует атрибут на `<html>`, а не на `<body>`)
  * Добавлен watcher для `theme` в Vue.js для автоматического применения темы при изменении
  * Реализовано сохранение выбранной темы в `localStorage` для персистентности между сессиями
  * Использованы иконки `fa-moon` (для светлой темы) и `fa-sun` (для темной темы) с динамическим переключением через `:class`
  * Добавлен динамический `title` для кнопки с подсказкой о переключении темы
  * Атрибут `data-bs-theme` перемещен с `<body>` на `<html>` и управляется через Vue.js (`:data-bs-theme="theme"`)
- **Функциональность**: Пользователь может переключать тему приложения между светлой и темной через кнопку с иконкой. Выбор темы сохраняется в localStorage и автоматически применяется при загрузке страницы. Bootstrap 5 корректно применяет тему ко всем компонентам через атрибут `data-bs-theme` на `<html>`
